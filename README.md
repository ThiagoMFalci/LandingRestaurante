# LandingRestaurante

Landing page estatica da Stratech para captar donos de restaurantes e levar os leads para o WhatsApp.

## Arquivos

- `index.html`
- `style.css`
- `script.js`

## Configurar WhatsApp

O WhatsApp atual esta configurado em `script.js` no formato internacional, somente numeros:

```js
const WHATSAPP_NUMBER = "5519982266218";
```

Depois, publique os arquivos em qualquer hospedagem estatica ou abra `index.html` no navegador.
